5  Cross-Company Comparison

Multi-company SDG disclosure benchmarking

Note

Status: Single-company mode. This chapter is structured to accommodate multiple companies. To add a new company:

  1. Run the Python pipeline on the company’s PDF reports
  2. Add the company slug to COMPANIES in R/load_evidence.R
  3. Re-render the book (quarto render)

All charts and tables below will auto-update with peer data.
