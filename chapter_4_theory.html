6  Longitudinal Theory Testing

SDG disclosure as a relational network — maturity transition analysis

6.1 Theoretical framework

Standard SDG disclosure analysis counts SDG mentions or scores disclosures on rubrics. This chapter extends that approach by treating SDG disclosure as a relational network — where SDGs are nodes connected to evidence types, implementation stages, and governance signals.

This allows us to examine not only which SDGs appear, but how they are embedded in operational, strategic, and measurable disclosure structures over time.

The core theoretical claim:

Firms transition through four stages of SDG disclosure maturity: rhetorical alignment → strategic commitment → operational implementation → governed, measured outcomes. This transition is visible not only in average scores but in the topological shift of the SDG-evidence network.
